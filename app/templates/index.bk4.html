How to Use SmartScrape
1. Upload Your CSV File

Upload a CSV file containing a 'Websites' column with the URLs you want to analyze.

2. Configure Analysis
  • Enter your OpenAI API key
  • Choose the GPT model
  • Provide analysis instructions
  • Optionally limit the number of rows to process
  • Add additional analysis columns if needed
3. Additional Columns

For each additional column:

  • Provide a column name
  • Write specific instructions for what GPT should analyze and output
  • You can add up to 5 additional columns
4. Processing

Click 'Start Processing' and wait for the analysis to complete. The results will automatically download as a CSV file.

About SmartScrape

SmartScrape is a powerful tool that combines web scraping with GPT analysis to help you gather and analyze website data efficiently.

Features:

  • Automated web scraping
  • GPT-powered content analysis
  • Custom analysis columns
  • Batch processing
  • CSV export
